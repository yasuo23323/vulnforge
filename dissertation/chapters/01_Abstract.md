## Abstract

Web application security scanning is a critical component of modern software development lifecycles, yet automated vulnerability scanners suffer from persistently high false positive rates that demand significant manual analysis effort. This dissertation presents VulnForge, an AI-enhanced web vulnerability scanner that integrates four industry-standard security tools - nuclei, sqlmap, dalfox, and ffuf - with Large Language Model (LLM) analysis to automatically identify and filter false positives.

The system employs a multi-strategy prompting approach, analyzing each scanner finding using three distinct techniques: Zero-shot prompting, Few-shot prompting with annotated examples, and Chain-of-Thought reasoning. These analyses are conducted automatically upon scan completion using the DeepSeek Chat API, with results stored for comparative evaluation. The system is deployed via Docker Compose for cross-platform compatibility.

Experimental evaluation on OWASP Juice Shop and DVWA targets demonstrates that VulnForge successfully detects over 100 findings across three scanners, with the LLM correctly identifying clear false positives such as IKEv2 and Dameng Database protocol detections as non-vulnerabilities. The three prompting strategies showed varying confidence levels, with Chain-of-Thought analysis demonstrating more nuanced judgement on ambiguous findings. Results indicate that multi-strategy LLM analysis can effectively reduce false positive noise in automated vulnerability scanning, though performance varies by finding type and prompting approach.
