# VulnForge - AI-Enhanced Vulnerability Scanner

## Quick Start

### Prerequisites
- Docker Desktop (Windows/Mac) or Docker Engine (Linux)

### Setup
1. Clone this repository
2. Copy .env.example to .env and set your LLM API Key
3. Run: docker compose up
4. Open: http://localhost:3000

### Usage
1. Enter a target URL
2. Select scanners
3. View results with LLM false-positive filtering
